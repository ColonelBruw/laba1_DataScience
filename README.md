## Лабораторная работа №1 по DataScience
# Анализ датасета проданых автомобилей
## *Выполнил: Чугунков В. С.*

**Цель:** 
- Освоить основные инструменты обработки и анализа данных инструментами языка Python (библиотеки pandas, seaborn, matplotlib, scipy) на примере **датасета содержащего информацию о проданных автомобилях**

**Задачи:**
1. Провести разведочный анализ данных
    - Собрать первичную информацию о датасете с помощью функций pandas head(), tail(), info(), describe()
    - Определить пустые/поврежденные значения ячейек и дубликаты строк
2. Провести предварительную обработку данных
    - Удалить дубликаты
    - Заменить/удалить некорректные значения
3. Представить содержимое датасета в виде графиков (гистограм, боксплотов, тепловых карт)
4. Посчитать коэффиценты Спирмена и Пирсона для 2 наиболее коррелирующих столбцов
5. Сделать выводы о датасете
    - Какая марка автомобилей наиболее распространена?
    - Какой тип автомобилей наиболее распространен?
    - Какой вид топлива наиболее распространен?
    - Какие поля больше всего коррелируют друг с другом?
